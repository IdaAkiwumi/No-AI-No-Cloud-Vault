### Why use this?
In 2026, privacy is a luxury. This tool ensures that your cloud provider only sees encrypted gibberish, while you maintain full usability with the "Secure Peek" feature.

### How to use:
1. Run `main.py`.
2. Encrypt any sensitive file.
3. Move the `.vault` file to your Cloud Drive.
4. When you need to see it, use "Secure Peek" to view it without leaving a trace.
